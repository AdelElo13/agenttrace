AgentTrace: decision-level observability for AI agents.

Not another LLM call logger. It shows which agent decisions were worth the money — and which were waste.

Every session: auto-traced.

Open source. MIT.

## X/Twitter (thread follow-up 1)

How it works:

1. Install with `agenttrace init`
2. Claude Code auto-traces every session
3. See: cost per agent, waste %, recommendations
4. Apply a fix → next sessions prove it worked

Built because I was spending $200/mo on Claude with zero visibility.

42% of the session was waste. Each fix: trivial.
